DOCKER

Docker

  • Ambiente que podamos replicar
  • Contenedor
    • Guardar cosas
    • Es portable
    • Interfaces claras
    • Lo podemos obtener o descargar de una ubicación remota.
  • Los servidores se relaciona a través de los puertos que tenemos abiertos
  • ¿Docker es una máquina virtual?
    • Es más eficiente en el manejo de los recursos
    • La principal diferencia es que en la máquina virtual se ve un hypervisor, necesita un sistema operativo para operar, en el contenedor no tenemos un sistema operativo completo, los recursos se comparten a traves de puertos en los containers
  • Elementos a tener en claro
    • Dockerfile
    • Docker container
    • Docker platform
    • Docker engineer
    • Docker daemon
    • Doker client
    • Docker registry
    • Docker repository
    • Docker volumes
    • Docker swarm
    • Docker compose
  • Interactuamos con docker por medio de nuestra línea de comandos o CLI
  • Comandos:
    • sudo yum update -y
    • sudo yum install docker
    • sudo service docker start
    • sudo usermod -a -G docker ec2-user
    • docker info
    • docker version
